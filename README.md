# PasswordGenerator
Pembuat Password yang aman dengan mudah.

# Fitur
Password dapat digenerate dengan beberapa opsi. Dengan adanya checkbox, user dapat memilih fitur yang akan digunakan pada saat hendak melakukan generate password

| Pilihan Fitur                                                              | Ket. |
|---------------------------------------------------------------|---|
| Upper Case                                               | ✔ |
| Lower Case                                            | ✔ |
| Symbol                                                  | ✔ |
| Number                                                | ✔ |
